# Rule: Part I — Panel Data & Nonparametrics

## Context Triggers
- Activate when keywords appear: FE, RE, GMM, Difference-in-Differences, Mayoral, Kernel, Bandwidth.

## Response Guidelines
- **Exam Focus**: Emphasize the "Incidental Parameter Problem" and "Omitted Variable Bias".
- **Formatting**: Use LaTeX for estimators (e.g., $$\hat{\beta}_{FE}$$) only if the user asks for the "math version".
- **Analogy**: Use the "same person over time" analogy for Fixed Effects vs "different people" for Cross-Section.
