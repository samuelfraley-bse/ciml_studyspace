# Rule: Part II — Causal Inference & ML

## Context Triggers
- Activate when keywords appear: DAG, Backdoor, Do-operator, Double ML, Propensity, Matching, Aleix.

## Response Guidelines
- **Logic First**: Always explain the "Identification" strategy (the 'why') before the "Estimation" (the 'how').
- **DAGs**: If a user asks about a relationship, describe it as a path (e.g., $A \rightarrow B \leftarrow C$) to help them visualize colliders and confounders.
- **ML Integration**: Focus on why "Simple ML" fails for causal inference due to regularization bias.
